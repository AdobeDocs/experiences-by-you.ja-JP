# データを活用してインパクトのあるストーリーを語る

データストーリーテリングは、データ、ビジュアライゼーションおよびナラティブを使用して、アートとサイエンスを結び付ける場所です。  これらのコンポーネントを活用することで、インパクトのあるデータストーリーを作成するには、次の3つの部分で構成されています。 データを使って効果的にストーリーを語ることで、[!DNL Analytics]はより多くのオーディエンスに親しみやすくなり、データ主導の意思決定を通じて組織にもたらす価値を高めることができます。

## 機会や課題の特定

まず、現在の状況を把握し、問題と機会を特定します。 このステージでは、オーディエンスを惹きつけるための十分な情報を提供できますが、その解決策を完全に明らかにすることはできません。 この紹介は非常に重要ですが（難しい場合もあります）、それはあなたの分析の動機です！  data storytellingの値は、**ビヘイビアーを変更した**&#x200B;結果のアクションを促すことです。 Data, Insight &amp; Analysisは無力です。

**[!DNL Adobe]Workspace ヒント：**&#x200B;問題や現在の状況を正確に示すために、質問の形式でビジュアライゼーションのタイトルを付けます。

## データを通じて説明する

問題を特定したら、データを利用して、問題の原因や機会を明らかにします。 目的を達成するための包括的な分析や検討ではなく、結果と、現在の状況を最もよく説明できる変数に焦点を当てます。  独立した変数と依存する変数の関係を示すのに役立つ魅力的なビジュアルを提供しますが、オーディエンスが簡単に利用できるように、シンプルなものにします。

**[!DNL Adobe]Workspace ヒント：**
共有するデータの種類に合わせて、シンプルで読みやすいビジュアルを作成します。

* カテゴリ データとバイナリ データに棒グラフを使用する
* 数値データに折れ線グラフ （または[!DNL Adobe]以外のバブル）を使用
* 統計的な関係を示すUSe散布図

## 解決策の提供

推奨されるアクションと、その目的を伝える。  関係者が、コストへの影響や変更に必要なリソースなど、十分な情報にもとづいて意思決定できるように、十分な情報を提供することが重要です。 これを実現する最も効果的な方法は、少なくともひとつ以上の重要なKPIを通じて、考えられる影響を定量化することです。

**[!DNL Adobe]Workspace ヒント：**&#x200B;分類や計算指標などのツールを活用して、[!DNL Adobe Analytics]内で考えられる影響を直接定量化し、[!DNL Adobe] Workspace内で要約数や変更点を使用して高音階で終了します。
