# 利害関係者の管理によるMarketo Engageの実装

Marketo Engageの導入は、MarTech スタックを拡大する重要な瞬間です。 マーケティングから販売、IT まで、様々な関係者を巻き込む必要があります。 適切な質問をし、定期的に必要な更新やサポートを伝えることで、新しいMarketo Engageインスタンスに対するサポートを組織から得る方法を説明します。 チュートリアルとテンプレート（ダウンロード可能なバージョンを含む）を使用すると、実装とユーザーのオンボーディング全体を通して内部通信をガイドできます。

## Marketo Engageの実装において関係者やリーダーと関わる方法

### 行動 1 - リーダーシップや利害関係者と連携して、Marketo Engageの能力を目標につなげる
