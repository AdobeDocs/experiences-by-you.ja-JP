# Marketo Engageを導入するための関係者の管理

Marketo Engageの導入は、マーテクスタックを成長させる重要な機会です。 マーケティングから営業、ITまで、さまざまな関係者を巻き込む必要があります。 適切な質問をし、アップデートやサポートを定期的に連絡することで、新しいMarketo Engageインスタンスに対する企業からのサポートを受ける方法をご紹介します。 チュートリアルとテンプレート（ダウンロード可能なバージョンを含む）を使用して、実装とユーザーオンボーディングを通じて社内コミュニケーションを導きます。

## Marketo Engage導入のために関係者や経営陣と連携する方法

### 行動1 - Marketo Engageの能力を目標に結び付けるために、経営陣と関係者の連携を図る

実装を始めるには、まず、経営陣やマーケティング部門および営業部門の関係者の優先事項を把握し、その優先順位をMarketo Engageがどのように実現するのかを教育する必要があります。 ここでは、チームリーダーや関係者とのコミュニケーションに役立つ質問をいくつか紹介します。

### 行動4 – 継続的なコミュニケーションの促進

実装プロセス全体を通じて、関係者との定期的なコミュニケーションチャネルを維持します。 チェックインミーティングのスケジュールを設定したり、提供されたメールテンプレートを使用して進捗状況を更新したりすることで、顧客に常に情報を提供して関与を促すことができます。 また、懸念事項や質問に迅速に対処し、チームからの継続的な賛同を確保できます。
